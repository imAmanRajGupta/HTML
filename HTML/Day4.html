Hyperlink

Hyperlink is used to link or attach different document to our web/html page.

Step to create Hyperlink

1. First we have to mark the content. to mark the content we use anchor tag. () It is container Tag.

2. Mention the path or location or hyper reference where we want to user to be redirect. We use href attribute.

Syntax: Google

Target Attribte:

Target attribute is used to specify where to open the link.

1._self: open in same browser tab.(default)

2._blank: open in new browser tab.

States of Anchor Tag

1. Not visited : sky blue

2. Active : red

3. Visited : Purple
